## 들어가며
### 준비물
#### JDK 설치 및 JAVA_HOME 환경변수 설정
스프링 버전에 따라 필요한 최소한의 자바 버전은 다음과 같다.

* 스프링4, 4.1 버전: 자바6 또는 그 이상
* 스프링3, 3.1, 3.2 버전: 자바5 또는 그 이상

이 책의 예제 코드는 다음을 기준으로 작성했다.

* 스프링 4.1
* 자바7
* 서블릿3/JSP2.2(톰캣7 버전 기준)

버전을 내리고 싶다면 자바7 버전에 맞게 작성한 코드를 자바6 버전에 수정한 후 컴파일하면 된다.

#### MAVEN 설치
스프링 프레임워크는 다양한 모듈을 포함하고 있고, 이들 모듈은 각 모듈마다 스프링 프레임워크에 포함되어 있지 않은 다른 모듈을 필요로 한다.

스프링2.5 버전까지는 스프링 프레임워크의 자체 모듈과 이들 모듈이 필요로 하는 모든 외부 모듈을 한 파일로 묶은 배포판을 제공했으나 스프링3 버전부터는 이러한 배포판을 제공하지 않는다. 대신 **메이븐 리포지토리**를 통해서 필요한 외부 모듈을 선택적으로 사용할 수 있도록 했다.

#### Eclipse 설치
책을 쓰는 시점에서 이클립스 최신 버전은 4.4.1이며, 메이븐 플러그인이 이미 연동되어 있고 웹 개발에 필요한 기능을 제공하는 Java EE Developers 버전을 다운받는다(다른 버전의 경우 메이븐 및 웹 개발 연동이 안 되어 있다).

참고로 이클립스 Kepler(4.3)이나 Luna(4.4)의 Java EE Developers 배포판은 이클립스 메이븐 플러그인이 이미 연동되어 있다. 만약 이클립스 메이븐 플러그인(m2e)이 설치되어 있지 않은 이클립스를 사용중이라면 다소 복잡한 과정을 거쳐야 할 수도 있다.

이 책의 예제는 UTF-8 인코딩을 이용해서 작성했다. 이클립스의 기본 인코딩 설정을 UTF-8로 변경[1]해놓으면 예제로 제공하는 코드를 이클립스에 임포트할 때마다 프로젝트의 인코딩을 변경해야 하는 불편함을 겪지 않아도 된다.

- - -
1. [Window] → [Preferences] → [General/Workspace] → Text file encoding → UTF-8
